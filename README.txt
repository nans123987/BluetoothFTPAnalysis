STEPS AND INSTRUCTIONS TO RUN THE PROJECT

1. TURN ON THE WIFI OF BOTH THE DEVICES(SENDER AND RECEIVER DEVICE).
2. INSTALL THE .apk FILE LOCATED IN THE bin FOLDER OF THE PROJECT DIRECTORY.
3. COPY AND PASTE THE FILES TO THE INTERNAL ROOT MEMORY OF THE PHONE(NOTE: NOT SD CARD), THAT NEEDS TO BE SHOWN IN THE APPLICATION.
4. RUN THE APPLICATION ON BOTH THE DEVICES.
5. SWITCH ON THE BLUETOOTH USING THE RESPECTIVE BUTTON IN THE APPLICATION.
6. CLICK ON GET PAIRED DEVICES TO GET THE LIST OF PAIRED DEVICES.
7. SELECT A DEVICE IN RANGE TO TRANSFER THE FILE.(NOTE: PAIR THE DEVICES USING NATIVE BLUETOOTH OF PHONE)
